# Awesome Knowledge Base OpenClaw 🦞

一个持续更新的 OpenClaw 知识库，收录原创书籍、实战经验、最佳实践与社区文章，希望能帮助更多人快速上手并实践使用 OpenClaw。

---



<br>



## 关于本仓库 About

`awesome-knowledge-base-openclaw` 是一个围绕 **OpenClaw** 展开的开源内容项目，目标是：

- ✍️ 编写一本系统性的 OpenClaw 原创书籍，覆盖从入门到进阶的完整知识体系
- 🛠️ 沉淀经过验证的最佳实践，帮助大家少走弯路
- 📝 收录高质量的技术文章与社区分享
- 🤝 建立一个开放、可持续贡献的中文OpenClaw 知识库



<br>



## 📚 《OpenClaw绿皮书》

| 章节                  | 内容                                   |
| --------------------- | -------------------------------------- |
| 🌱 **第一章 · 初识篇** | 什么是 OpenClaw & 为什么火爆           |
| 💡 **第二章 · 点亮篇** | 部署方案 & 各 OS 安装 & 初始配置       |
| ⚡ **第三章 · 能力篇** | 模型、渠道、技能、搜索、记忆等核心能力 |
| 🚀 **第四章 · 进阶篇** | 常用命令、多智能体、实战场景           |
| 🛡️ **第五章 · 建议篇** | 安全、成本、心态三大建议               |
| 📖 **第六章 · 概念篇** | 智能体定义 & 相关术语                  |
| 📎 **附录**            | FAQ 常见问题 & 扩展阅读                |
